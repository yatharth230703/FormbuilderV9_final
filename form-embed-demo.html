Form Embed Demo

This page demonstrates how to properly embed and resize your form iframe.

How to use:

  1. Include the form-embed-snippet.js script on your page
  2. Add the data-form-embed attribute to your form iframe
  3. The iframe will automatically resize to fit the content

HTML Code Example:

<!-- Include the form embed script --> <script src="form-embed-snippet.js"></script> <!-- Add your form iframe with data-form-embed attribute --> <iframe data-form-embed="true" src="http://your-form-url.com/embed?form=1" frameborder="0" style="width: 100%;"> </iframe>

JavaScript Configuration:

// Optional: Configure the form embed script document.addEventListener('DOMContentLoaded', function() { // Enable debug mode window.FormEmbed.config.debug = true; // Set minimum height (default: 600px) window.FormEmbed.config.minHeight = 500; // Set height buffer to prevent scrollbars (default: 10px) window.FormEmbed.config.heightBuffer = 20; // Set transition duration for smooth height changes (default: 300ms) window.FormEmbed.config.transitionDuration = 200; // Initialize (this happens automatically, but you can call it manually) window.FormEmbed.initialize(); });
